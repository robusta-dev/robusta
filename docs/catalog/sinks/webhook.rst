Webhook 
###########

Robusta can send playbooks results to a webhook.

.. admonition:: values.yaml

    .. code-block:: yaml

        sinks_config:
        - webhook_sink:
            name: webhook_sink
            url: "https://my-webhook-service.com/robusta-alerts"
